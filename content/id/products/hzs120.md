## Gambaran Umum

HZS120 adalah pabrik batching beton skala besar yang mampu memproduksi 120 m³ beton berkualitas tinggi per jam. Ini adalah pilihan utama untuk proyek infrastruktur besar termasuk jalan raya, bandara, dan pengembangan komersial besar.

## Fitur Utama

- **Mixer kuat JS3000** — keluaran tinggi dengan kualitas yang konsisten
- **Sistem pemanas agregat** — opsional untuk iklim dingin
- **Kontrol slump otomatis** — umpan balik sensor kelembaban
- **Integrasi ERP** — manajemen data produksi
- **Pengiriman kontainer** — cocok untuk kontainer 40ft standar

## Aplikasi

- Proyek jalan raya dan kereta api
- Konstruksi landasan pacu bandara
- Kompleks komersial besar
- Infrastruktur besar (bendungan, pelabuhan, jembatan)
- Pabrik ready-mix besar
